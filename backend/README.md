# LinkedIn Clone (Full Stack MERN Project)

This is a simple LinkedIn-style social feed application where users can:

- Signup / Login (JWT Auth)
- Create Posts (Text + Image Upload)
- Like / Unlike Posts
- Comment on Posts
- Edit and Delete 
- View all posts
